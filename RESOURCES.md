# TypeScript Resources

## Knowledge

- [The TypeScript Handbook](https://www.typescriptlang.org/docs/handbook/intro.html)
  Official, authoritative, and beginner-friendly. Use for: ground-truth syntax and semantics for any type feature.
- [TypeScript Playground](https://www.typescriptlang.org/play)
  Official in-browser editor showing live type errors and compiled JS output. Use for: experimenting with snippets from lessons, checking "what does this type actually do."
- [JavaScript basics — MDN Web Docs](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Guide)
  Since the user is also new to JS, this is the trusted reference for core JS concepts (variables, functions, objects, arrays) that TypeScript builds on.
- [State of JavaScript 2024 — Usage](https://2024.stateofjs.com/en-US/usage/)
  Survey data showing TypeScript is now the majority default for production code (most devs write more TS than JS). Use for: justifying the TS-primary, JS-as-needed teaching order (see learning-records/0003).

## Wisdom (Communities)

- [r/typescript](https://reddit.com/r/typescript)
  Active community for questions, code review, and discussion of patterns. Use for: "is this AI-generated type idiomatic?" style questions once fundamentals are solid.
